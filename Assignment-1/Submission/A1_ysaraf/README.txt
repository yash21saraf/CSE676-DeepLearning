Instructions to run the code -

- Run jupyter notebook. 
- Run cells in sequential order. 
- For graphs a log file will be created in the same folder in the summaries. 
- Open tensorboard using command. 
	tensorboard --logdir=summaries/

- The training and testing graphs can be seen.
